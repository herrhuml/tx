# TXLogger
> TX logs made easy.

A small app to make the process of making logs easier.

## Installation
[download]

```
1. download the latest release
2. extract
3. run
```

## Usage

1. enter the IDs
2. press the 'Get Info' button
3. edit anything you need
4. press the 'Create Log' button and you'll get the log
